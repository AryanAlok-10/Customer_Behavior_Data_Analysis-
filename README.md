# Customer Behavior Data Analysis

## 📌 Project Overview
This project focuses on analyzing complex customer behavior datasets to uncover actionable insights and solve real-world market challenges.
By leveraging an end-to-end data analytics workflow, the project transitions raw data into strategic business recommendations, 
culminating in an executive presentation for key stakeholders.

---

## 🛠️ Tech Stack & Tools
* **Python**: Used for Data Cleaning, Handling Missing Values, and Exploratory Data Analysis (EDA).
* **SQL**: Used for deep-dive querying, data aggregation, and performance benchmarking.
* **Power BI**: Used to build a dynamic, interactive dashboard for trend visualization.
* **Microsoft PowerPoint**: Used to synthesize findings into a final executive-ready deck.

---

## 📈 Key Outcomes & Insights
* **Market Problem Solving**: Addressed critical business questions regarding customer retention, purchasing patterns, and engagement trends.
* **Interactive Dashboards**: Delivered a Power BI dashboard allowing stakeholders to filter data dynamically and track KPIs in real time.
* **Strategic Recommendations**: Provided data-backed recommendations focused on optimizing customer experience and driving business growth.

---

## 📂 Project Structure
* `/data`: Raw and cleaned datasets used during the analysis.
* `/scripts`: Python notebooks and SQL queries used for data processing.
* `/dashboards`: Power BI file (`.pbix`) containing the interactive visualizations.
* `/presentation`: The final PPT presentation detailing findings and outcomes.
